# Daily_Learning_Log

日々の学習記録をまとめたリポジトリです。
プログラミング学習4ヶ月目に入り、ちいさく日記にて日々の学習を記録していきます。

## ▶️ このリポジトリについて
毎日の TIL（Today I Learned）を自動で記録しています。
学習内容・メモ・成果などを Markdown 形式で保存しています

## ▶️ 記録内容
- その日の学習内容
- AI生成の振り返り
- 今日のメモ
- 気づき、モチベーション
- 進捗状況

## ▶️ ファイル命名規則
- `yymmdd_til.md`
- 例： `250627_til.md`（2025年6月27日の記録）

## ▶️ 目的
学んだ内容を蓄積し、成長の軌跡を確認できるようにしています。
